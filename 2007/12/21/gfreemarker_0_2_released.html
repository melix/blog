GFreeMarker 0.2 released

21 December 2007

Tags: freemarker groovy java

I have released a new version of GFreeMarker, an integration of the FreeMarker template engine in Groovy. This release has no functional change, but has been refactored :

  • Built made by Maven 2

  • Compatibility with JDK 1.4

  • Uses Groovy 1.5
